Frequently Asked Questions

How does VeilCast hide windows from screen sharing?

VeilCast uses advanced native Windows architecture to hook directly into the display rendering pipeline, making specific windows mathematically invisible to capture software like Zoom, Teams, and OBS.

Does VeilCast collect my private data?

No. VeilCast is a 100% offline, native application. It does not contain telemetry, internet connectivity, or analytics. Your data never leaves your computer.
